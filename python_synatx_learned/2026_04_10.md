## 1. String 排序

### Python
```python
sorted("eat")              # → ['a', 'e', 't']  返回 list
"".join(sorted("eat"))     # → "aet"  拼成 string
```

### JS/TS
```ts
"eat".split("").sort().join("")  // → "aet"
```

## 2. dict / Map 的 get + 默认值

### Python
```python
d.get(key, [])   # key 不存在时返回 []，但不会存进 dict
```

### JS/TS
```ts
map.get(key) || []   // key 不存在时用 []
```

## 3. dict / Map 的条件写入

三种等价写法（Python）：

```python
# 写法 1：get + 创建新 list
d[key] = d.get(key, []) + [val]

# 写法 2：if 判断
if key not in d:
    d[key] = []
d[key].append(val)

# 写法 3：setdefault（最 Pythonic）
d.setdefault(key, []).append(val)
```

> `setdefault(key, default)`：如果 key 不存在，先存入 default 并返回它；如果存在，直接返回已有的值。

### JS/TS 对应写法
```ts
if (!map.has(key)) map.set(key, [])
map.get(key)!.push(val)

// 或者
map.set(key, [...map.get(key) || [], val])
```

## 4. 为什么 append 不能直接赋值

```python
# ✗ 错误：append 返回 None，赋值后 d[key] 变成 None
d[key] = d.get(key, []).append(val)

# ✓ 正确：用 + 返回新 list 再赋值
d[key] = d.get(key, []) + [val]
```

**原则：** Python 原地修改的方法返回 `None`（`append`, `sort`, `update`）；返回新值的操作不修改原对象（`+`, `sorted()`, `str.replace()`）

## 5. 取出 Map / dict 所有 value

### Python
```python
list(d.values())   # → [[...], [...], [...]]
```

### JS/TS
```ts
Array.from(map.values())   // → [[...], [...], [...]]
```

> `map.values()` 返回的是 Iterator，不是数组，需要用 `Array.from()` 转换。
