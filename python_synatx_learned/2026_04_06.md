## From: 001 Two Sum

### enumerate()

遍历 list 同时获取 index 和 value：

```python
for index, num in enumerate(nums):
```

不能直接 `for index, num in nums`，那样会报错。

### dict 常用 API

| 操作 | 写法 |
|------|------|
| 写/更新 | `d[key] = value` |
| 读 | `d[key]`（不存在抛 KeyError）|
| 读（安全） | `d.get(key)`（不存在返回 None）|
| 读（带默认值） | `d.get(key, default)` |
| 查 | `key in d`（返回 bool） |
| 删 | `del d[key]` / `d.pop(key)` / `d.pop(key, default)` |

### 判断 key 是否存在：用 `in` 而不是 `get`

```python
# 有 bug：当 value 是 0 时，if 0 为 falsy，会误判为不存在
if dict.get(key):

# 正确写法
if key in dict:
```

`get` 返回值是 falsy（`None`, `0`, `""`, `[]`）时都会误判，`in` 只判断 key 存不存在，没有歧义。

## From: 027 Remove Element

### 多变量赋值

```python
# 正确
slow, fast = 0, 0

# 错误：会报 TypeError: cannot unpack non-iterable int object
slow, fast = 0
```

### 没有 `++` 运算符

Python 不支持 `++`，用 `+=` 代替：

```python
slow += 1
```

### `+=` 和 `=+` 的区别

```python
a = 5
a += 1  # a = a + 1 → 6
a =+ 1  # a = (+1) → 1，直接覆盖
```

`=+` 不是运算符，会被解析为 `=` 加一元 `+`。

### 切片获取子数组

Python 和 JS 的切片都**返回新数组**，不修改原数组：

```python
a = [1, 2, 3, 4]
b = a[:2]   # b = [1, 2]，a 不变
```

```js
const a = [1, 2, 3, 4];
const b = a.slice(0, 2);  // b = [1, 2]，a 不变
```

### 原地截断数组

JS 用 `splice`（注意不是 `slice`）：

```js
a.splice(2);  // a 变成 [1, 2]，原地修改，返回被删除的 [3, 4]
```

Python 用切片赋值——选中一段，用右边的列表替换掉：

```python
a = [1, 2, 3, 4]
a[2:] = []      # 把 index 2 到末尾的 [3, 4] 替换成 []，a 变成 [1, 2]
a[2:] = [8, 9]  # 把 index 2 到末尾替换成 [8, 9]，a 变成 [1, 2, 8, 9]
```
