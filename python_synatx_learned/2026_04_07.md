# Python Sort, Lambda, Array Slicing

## Sort

```python
# 原地排序，修改原数组，返回 None
nums.sort()

# 返回新列表，不改原数组
new_list = sorted(nums)

# 自定义排序规则，用 key 参数
intervals.sort(key=lambda x: x[0])              # 原地，按第一个元素升序
sorted_intervals = sorted(intervals, key=lambda x: x[0])  # 新列表

# 降序
nums.sort(reverse=True)
sorted(nums, reverse=True)

# 默认排序：升序。嵌套列表先按第一个元素比，相同再按第二个
sorted([[2,6],[1,3],[1,1]])  # [[1,1],[1,3],[2,6]]
```

| | `sorted()` | `.sort()` |
|---|---|---|
| 返回值 | 新列表 | `None` |
| 原数组 | 不变 | 被修改 |

注意：`key` 是 keyword-only 参数，必须显式写出来：
```python
nums.sort(key=lambda x: x[0])     # 正确
nums.sort(lambda x: x[0])         # TypeError
```

### 多级排序

用元组实现，按顺序依次比较：

```python
people = [
    {"name": "Alice", "age": 30},
    {"name": "Bob", "age": 25},
    {"name": "Charlie", "age": 35}
]

# 按 age 排序
sorted(people, key=lambda x: x["age"])

# 多级排序：先按 age，相同再按 name
sorted(people, key=lambda x: (x["age"], x["name"]))

# 某个字段降序：数字取负
sorted(people, key=lambda x: (-x["age"], x["name"]))
```

### 复杂排序：cmp_to_key

当 lambda + 元组无法表达排序逻辑时（比如字符串降序 + 数字升序混合），用比较函数：

```python
from functools import cmp_to_key  # 必须 import

# 比较函数规则：
# 返回负数 → a 排前面
# 返回正数 → b 排前面
# 返回 0   → 相等

def compare(a, b):
    if a["age"] != b["age"]:
        return a["age"] - b["age"]      # age 升序
    if a["name"] < b["name"]:
        return 1                         # name 降序
    elif a["name"] > b["name"]:
        return -1
    return 0

sorted(people, key=cmp_to_key(compare))
```

### 实战：拼接最大数（LeetCode 179）

`cmp_to_key` 最经典的场景：给 `[3, 30, 34]`，拼成最大的数 `"34330"`。

排序依据是两个元素之间的拼接关系，无法用 `key` 给单个元素算出一个值，所以必须用比较函数。

```python
from functools import cmp_to_key

nums = [3, 30, 34]
nums_str = [str(x) for x in nums]

# 比较 a+b vs b+a，大的排前面
nums_str.sort(key=cmp_to_key(lambda a, b: -1 if a+b > b+a else 1))
# ["34", "3", "30"] → "34330"
```

比较过程：
- `"3"` vs `"30"`：`"330"` vs `"303"` → `"330"` 大 → `"3"` 排前面
- `"3"` vs `"34"`：`"334"` vs `"343"` → `"343"` 大 → `"34"` 排前面
- `"34"` vs `"30"`：`"3430"` vs `"3034"` → `"3430"` 大 → `"34"` 排前面

总结：
- **简单排序** → `key=lambda x: x["age"]`
- **多级排序** → `key=lambda x: (x["age"], x["name"])`
- **复杂逻辑** → `cmp_to_key(compare)`

## Lambda

匿名函数，一行简写。

语法：`lambda 参数: 返回值`

```python
# lambda 写法
f = lambda x: x[0]

# 等价于
def f(x):
    return x[0]

# 更多例子
lambda x: x * 2           # 输入x，返回x*2
lambda a, b: a + b         # 两个参数，返回a+b
lambda x: (x[1], x[0])    # 返回元组，可用于多级排序
```

## Array Slicing

语法：`list[start:end:step]`

- `start` 起始索引（包含），默认 0
- `end` 结束索引（不包含），默认到末尾
- `step` 步长，默认 1

```python
nums = [0, 1, 2, 3, 4, 5]

nums[1:]      # [1, 2, 3, 4, 5]   从索引1到末尾
nums[:3]      # [0, 1, 2]          从开头到索引3（不包含）
nums[1:4]     # [1, 2, 3]          索引1到4（不包含）
nums[-1]      # 5                   最后一个元素
nums[-2:]     # [4, 5]             最后两个元素
nums[::2]     # [0, 2, 4]          每隔一个取
nums[::-1]    # [5, 4, 3, 2, 1, 0] 反转

# 注意：[1:0] 是空切片，因为 start >= end
```

## 逻辑运算符

Python 用英文单词，不用符号：

| Python | JS/TS |
|--------|-------|
| `and` | `&&` |
| `or` | `\|\|` |
| `not` | `!` |

优先级：`not` > `and` > `or`

```python
a or not b        # 等价于 JS: a || (!b)
(a or b) or (c and d)  # 等价于 JS: (a || b) || (c && d)

# 实际使用
while i < length and intervals[i][1] < newInterval[0]:
    ...

if not found or (x > 0 and y > 0):
    ...
nums[1:0]     # []
```
