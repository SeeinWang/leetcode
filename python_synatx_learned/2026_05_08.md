## for loop 中 range 的写法

**正序**
```python
for i in range(n):          # 0, 1, 2, ..., n-1
for i in range(1, n):       # 1, 2, ..., n-1
for i in range(1, n-1):     # 1, 2, ..., n-2
```

**倒序**
```python
for i in range(n-1, -1, -1):    # n-1, n-2, ..., 0
for i in range(n-2, -1, -1):    # n-2, n-3, ..., 0
for i in range(n-1, 0, -1):     # n-1, n-2, ..., 1
```

**步长**
```python
for i in range(0, n, 2):        # 0, 2, 4, ... (偶数)
for i in range(1, n, 2):        # 1, 3, 5, ... (奇数)
```

规律：`range(start, stop, step)`，stop 是**不包含**的边界，倒序时 step 为负数。

## dict 的常用操作

```python
d = {"foo": [1, 2, 3]}

# 检查 key
"foo" in d               # True（默认检查 key）
"foo" in d.keys()        # 同上

# 检查 value
[1, 2, 3] in d.values()  # True

# 检查 key-value 对
("foo", [1, 2, 3]) in d.items()  # True
```
