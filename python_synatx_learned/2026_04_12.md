## 1. 检查元素是否在 set / list 中

### Python
```python
num in my_set    # True/False，set 是 O(1)
num in my_list   # True/False，list 是 O(n)
num not in my_set
```

### JS/TS
```ts
set.has(num)        // Set，O(1)
arr.includes(num)   // Array，O(n)
```

## 2. 集合运算

### Python
```python
set(a) & set(b)   # 交集 ∩
set(a) | set(b)   # 并集 ∪
set(a) - set(b)   # 差集
```

### JS/TS
JS 没有内置运算符，需要手动实现：
```ts
const setB = new Set(b)
a.filter(n => setB.has(n))   // 交集
```

## 3. 推导式（Comprehension）

### 字典推导式
```python
d = {k: v for k, v in some_list}

# 例子
d = {i: i*2 for i in range(5)}
# → {0: 0, 1: 2, 2: 4, 3: 6, 4: 8}
```

### 列表推导式
```python
[i*2 for i in range(5)]   # → [0, 2, 4, 6, 8]
```

### 集合推导式
```python
{i*2 for i in range(5)}   # → {0, 2, 4, 6, 8}
```

### JS/TS 对应写法
```ts
// 字典（对象）
Object.fromEntries(arr.map(([k, v]) => [k, v]))

// 列表
arr.map(x => x * 2)

// 集合
new Set(arr.map(x => x * 2))
```

### 推导式 = map + filter 合体
```python
[i*2 for i in range(5) if i > 2]   # Python

arr.filter(i => i > 2).map(i => i * 2)   // JS 需要分开写
```

### LeetCode 常见用法
```python
# list 转 dict（值 → 下标）
{v: i for i, v in enumerate(nums)}

# 过滤
[x for x in nums if x > 0]

# 矩阵展平
[x for row in matrix for x in row]
```
