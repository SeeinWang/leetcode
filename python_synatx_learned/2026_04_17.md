## 1. 遍历索引的几种写法

### 只要索引
```python
for i in range(len(s)):
    print(s[i])
```

### 只要字符
```python
for ch in s:
    print(ch)
```

### 同时要索引和字符
```python
for i, ch in enumerate(s):
    print(i, ch)
```

> 如果只用索引，不用字符，直接用 `range(len(s))` 更简洁，不要用 `enumerate` 再丢掉字符。

### JS/TS 对应写法
```ts
// 只要索引
for (let i = 0; i < s.length; i++) { ... }

// 只要字符
for (const ch of s) { ... }

// 同时要索引和字符
for (const [i, ch] of [...s].entries()) { ... }
```

## 2. 类内方法互相调用必须加 `self.`

### 错误写法
```python
class Solution:
    def expand(self, s, l, r):
        ...

    def longestPalindrome(self, s):
        odd = expand(s, 0, 0)   # ❌ NameError: expand is not defined
```

### 正确写法
```python
class Solution:
    def expand(self, s, l, r):
        ...

    def longestPalindrome(self, s):
        odd = self.expand(s, 0, 0)   # ✅
```

> 类里定义的函数叫**方法**，不是全局函数。必须通过 `self.方法名()` 调用。

### 如果不需要实例状态：用 `@staticmethod`
```python
class Solution:
    @staticmethod
    def expand(s, l, r):   # 不用写 self
        ...

    def longestPalindrome(self, s):
        odd = Solution.expand(s, 0, 0)
        # 或者 self.expand(s, 0, 0) 也行
```

### JS/TS 对应
```ts
class Solution {
    expand(s: string, l: number, r: number) { ... }

    longestPalindrome(s: string) {
        const odd = this.expand(s, 0, 0)  // 必须加 this.
    }
}
```

## 3. 字符串切片：冒号，不是逗号

### 错误
```python
s[l+1, r]   # ❌ TypeError: string indices must be integers
```

### 正确
```python
s[l+1:r]    # ✅ 包头不包尾
```

### 常见切片
```python
s[a:b]      # [a, b)
s[:b]       # 从头到 b（不含）
s[a:]       # 从 a 到结尾
s[::-1]     # 反转
s[::2]      # 每隔一个取一个
```

### JS/TS 对应
```ts
s.slice(a, b)    // [a, b)
s.slice(0, b)
s.slice(a)
s.split('').reverse().join('')   // 反转
```

## 4. 别用内置类型名当变量名

### 坑
```python
for index, str in enumerate(s):   # ❌ str 覆盖了内置 str 类型
    ...
    len(str)   # 本意是 len(s)，结果 str 成了当前字符
```

常见需要避开的名字：`str`、`list`、`dict`、`set`、`tuple`、`int`、`type`、`id`、`input`。

> 建议：字符用 `ch`、`c`，字符串用 `s`、`word`、`text`。

## 5. JS 数组方法在 Python 中的对应

### `every` → `all()`
```python
# JS: count.every(c => c === 0)
all(c == 0 for c in count)
```

### `some` → `any()`
```python
# JS: arr.some(x => x > 0)
any(x > 0 for x in arr)
```

### `map` → `map()` 或列表推导
```python
# JS: arr.map(x => x * 2)
list(map(lambda x: x * 2, arr))
# 或更 Pythonic：
[x * 2 for x in arr]
```

### `filter` → `filter()` 或列表推导
```python
# JS: arr.filter(x => x > 0)
list(filter(lambda x: x > 0, arr))
# 或更 Pythonic：
[x for x in arr if x > 0]
```

> `all()`、`any()`、`map()`、`filter()` 都是 Python 内置函数，不需要 import。
> `map()` 和 `filter()` 返回迭代器，需要 `list()` 转换才能得到列表。

## 6. `ord()` 和 `chr()`

`ord()` 只能传**单个字符**，传字符串会报错：

```python
ord('a')    # ✅ → 97
ord('ab')   # ❌ TypeError: ord() expected a character, but string of length 2 found
```

常见用法：

```python
# 字符映射到数组下标
count[ord(s[i]) - ord('a')] += 1

# 字符加减偏移
chr(ord('a') + 3)  # → 'd'
```

> `ord` 字符→数字，`chr` 数字→字符，都是内置函数。

## 7. 判断字符是否为字母或数字：`isalnum()`

```python
c.isalnum()   # 字母或数字返回 True，其他（空格、标点等）返回 False
```

替代手写范围判断：
```python
# 不需要这样写：
('a' <= c <= 'z') or ('A' <= c <= 'Z') or ('0' <= c <= '9')
```

